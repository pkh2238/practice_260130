
# Lotto Number Generator

## Overview

A simple, stylish, and interactive web application to generate random lottery numbers. This application will be built using modern web standards, including HTML, CSS, and JavaScript, leveraging Web Components for a modular structure.

## Project Outline

*   **index.html**: Main entry point of the application.
*   **style.css**: Modern CSS for styling the application, including a responsive design.
*   **main.js**: Core application logic, including the lottery number generation and the definition of the Web Component.

### Style and Design

*   **Layout**: A centered, clean layout that is easy to use.
*   **Typography**: Clear and readable fonts.
*   **Color Scheme**: A visually appealing color palette.
*   **Interactivity**: Smooth transitions and a clear visual response to user actions.

### Features

*   Generate 6 unique random numbers between 1 and 45.
*   Display the generated numbers in a visually appealing way.
*   A "Generate" button to trigger the number generation.
